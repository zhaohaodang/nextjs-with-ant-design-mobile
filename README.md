# nextjs-with-ant-design-mobile

## 简介

Next.js脚手架工具 `npx create-next-app` 会快速创建一个Next.js项目，在此基础上自定义一些配置，搭配 `antd-mobile` ，使项目支持以下特点。如果这些特点符合你的项目需求，建议你clone本仓库，进行二次开发。

## 特点

 - 同时支持 `less` 和 `css` 导入，也支持 `Css Modules`

 - 按需加载 `antd-mobile` 组件

 - 定制 `antd-mobile` 主题

## 版本信息

 `next: 10.0.1`

 `antd-mobile: ^2.3.4`
